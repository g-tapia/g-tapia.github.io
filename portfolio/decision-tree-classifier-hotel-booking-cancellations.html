Room Reservation vs Actual Room Assigned

Does the Comparison Between Reserved and Assigned Room Types Impact Cancellation Rates?

# Analyzing the relationship between room changes and cancellations without modifying the original dataframe
room_change_cancellation_analysis <- hotel_bookings %>%
  mutate(room_match = ifelse(assigned_room_type == reserved_room_type, "Matched", "Changed")) %>%
  group_by(room_match, is_canceled) %>%
  count() %>%
  ungroup() %>%
  mutate(freq = n / sum(n))

# Plot with percentage annotations
ggplot(room_change_cancellation_analysis, aes(x = room_match, y = freq, fill = factor(is_canceled))) +
  geom_bar(stat = "identity", position = position_dodge()) +
  scale_fill_manual(values = c("green", "red"), labels = c("Not Canceled", "Canceled")) +
  labs(title = "Room Change vs. Cancellation", x = "Room Change", y = "Frequency", fill = "Reservation Status") +
  geom_text(aes(label = scales::percent(freq, accuracy = 1)), 
            position = position_dodge(width = 0.9), 
            vjust = -0.5, size = 3)

Contrary to initial expectations, I uncovered a counterintuitive trend: a majority of reservations with matched room types (i.e., the room reserved and the actual room provided) indicated that customers who received the room they requested were more likely to cancel. This is surprising, as one would expect cancellations to be more common among customers who did not get their desired room. This finding prompts a further examination of other variables, as it challenges our initial assumptions.

# reference https://stackoverflow.com/questions/2547402/how-to-find-the-statistical-mode
find_mode <- function(x) {
  u <- unique(x)
  tab <- tabulate(match(x, u))
  u[tab == max(tab)]
}
